data-*

  • Store extra information on tags
  • Associated with element but does not need any defined meaning

Any attribute on any element whose attribute name starts with data- is a data attribute.

Html


<article
    id="myArticle"
    data-columns="3"
    data-index-number="12314">
    ...
</article>

JavaScript access

Dashes are converted to camelCase


const article = document.querySelector('#myArticle');

article.dataset.columns // "3"
article.dataset.indexNumber // "12314"

CSS access


article::before {
    content: attr(data-index-number);
}

article[data-columns='3'] {
    width: 400px;
}

Embed

iframe

Embeds another document within the current HTML document.

Use only when completely necessary.

  • Be aware of copyright
  • Be aware of security
    • sandbox
    • HTTPS
    • CSP (Content Security Policy)
    • X-Frame-Options

<iframe src="url" width="100%" height="500" allowfullscreen sandbox>
    <p><a href="url"> link </a></p>
</iframe>
sandbox for security (required)
