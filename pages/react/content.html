JSX

  • JavaScript XML
  • Properties and methods in JSX are camelCase.
  • className is used instead of class
  • Self-closing tags must end in a slash - e.g. <img/>

JavaScript expressions can also be embedded inside JSX using curly braces, including variables, functions, and properties.

Class Components

  • Regular components

A class component must include render(), and the return can only return one parent element. 


class ClassComponent extends Component {
  render() {
  	return <div>Example</div>
  }
}

Simple Components

  • It's a Function
  • Doesn't use the class keyword.

components can be nested in other components, and simple and class components can be mixed. 


const SimpleComponent = () => {
  return <div>Example</div>
}

if the return is contained to one line, it does not need parentheses.

props

The data that's stored here is known as the virtual DOM, which is a fast and efficient way of syncing data with the actual DOM.

keys

Keys help React identify which items have changed, are added, or are removed.

state

You can think of state as any data that should be saved and modified without necessarily being added to a database - for example, adding and removing items from a shopping cart before confirming your purchase. You must use this.setState() to modify an array. Simply applying a new value to this.state.property will not work.
