display: flex Element turns into a flex container
flex-direction
  • row - default
  • column
  • row-reverse
  • column-reverse

Flexbox: justify-content

  • justify items around main axis
  • for flex container
Main axis
horizontal flex-direction: row
vertical flex-direction: column
value
center Center, space on edges
flex-start
  • default
  • if flex-direction: row - aligns to left
  • if flex-direction: column - aligns to top
flex-end
  • if flex-direction: row - aligns to right
  • if flex-direction: column - aligns to bottom
space-between First and last pushed to edges
space-around Half space on first and last
space-evenly Full space evenly

Felxbox: align

  • aligns elements around cross axis

Cross axis
vertical flex-direction: row
horizontal flex-direction: column
property
align-items for flex container
align-self for flex items
value
center Center, space on edges
flex-start
  • if flex-direction: row - aligns to top
  • if flex-direction: column - aligns to left
flex-end
  • if flex-direction: row - aligns to bottom
  • if flex-direction: column - aligns to right
stretch
  • default
  • Fills the flex container
baseline

like flex-start but aligns text baseline

Flexbox: flex-wrap

  • Extra items move to new row or column
  • Break point depends on sizes
  • For flex container

nowrap default
wrap
  • row = wraps top to bottom
  • column = wraps left to right
wrap-reverse
  • row = wraps bottom to top
  • column = wraps right to left

Flexbox: flex

  • flex-grow + flex-shrink + flex-basis
  • for flex items
flex-grow
  • container > items
  • item grows
  • number value
flex-shrink
  • items > container
  • item shrinks
  • number value
flex-basis
  • sets the initial size
  • unit value
 

// For flex items

/* unitless number: flex-grow */
flex: 2;

/* width/height: flex-basis */
flex: 10em;
flex: 30%;
flex: min-content;

/* flex-grow | flex-basis */
flex: 1 30px;

/* flex-grow | flex-shrink */
flex: 2 2;

/* flex-grow | flex-shrink | flex-basis */
flex: 2 2 10%;

Flexbox: order

  • Order of how flex items appear in the flex container
  • Can have negative values
  • For flex items

.container {
    display: flex;
}
       
.box {
    order: 2;
}

grid-template-rows/columns

  • for grid container
number of parameters number of rows/cols
value of parameters width of each row/col
repeat(100, 50px) 100 rows/cols of 50px
repeat(auto-fill, 50px) fill with rows/cols of 50px
repeat(auto-fit, 50px) stretches rows/cols
minmax(50px, 200px) limit row/col size

example:


/* 3 columns of 90px - 1fr */
grid-template-columns: repeat(3, minmax(90px, 1fr));

/* fill with rows of 60px - 1fr */
grid-template-rows: repeat(auto-fill, minmax(60px, 1fr));

grid-row/column/area

  • for grid items
grid-row
  • rows an item will consume
  • grid-row: row1/row2
  • grid-row: 1/2
grid-column
  • columns an item will consume
  • grid-column: col1/col2
  • grid-column: 1/2
grid-area
  • rows & columns
  • grid-area: row1/col1/row2/col2
  • grid-column: 1/1/2/2
