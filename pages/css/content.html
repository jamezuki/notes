Syntax

selector = HTML element to style.


// comment
/* block comment */
		
selector1, selector2 {
    property: value;
}

Properties

Rules selector { property: value; }
Cascade The last rule is stronger.
Specificity The more specific the stronger.
Inheritance Rules can be inherited.

Override

  • browsers read CSS from top to bottom
  • ids overrides classes
  • inline styles overrides ids and classes
  • !important overrides everything

Lifecycle

  1. Browser converts HTML to DOM(Document Object Model).
  2. Browser loads the resources (CSS, JS, media).
  3. Browser attaches the style to DOM.
  4. Browser paints page.

How to use

Inline

style attribute inside HTML elements.

  • Single html element

<tag style="property:value;"> value </tag>

Internal

style tag in the head section.

  • Single page

<head>
    <style>
        body {background-color: powderblue;}
        h1 {color: blue;}
        p {color: red;}
    </style>
</head>

External

link tag in head section.

  • Multiple pages

file.html


<head>
    <link rel="stylesheet" href="style.css">
</head>

style.css


h1 {
    color: blue;
}

margin collapse

The top and bottom margins of blocks are sometimes combined (collapsed) into a single margin whose size is the largest of the individual margins.

Note that the margins of floating and absolutely positioned elements never collapse.

Gradients

linear-gradient

dir, color1 [color-stop], color2, ...


/* top to bottom, blue 0%, pink (default) */
linear-gradient(blue, pink);

linear-gradient(70deg, blue, pink);

/* starts at opposite side, orange stops at 60% */
linear-gradient(to bottom right, orange 60%, cyan);

radial-gradient

shape position, color1 [color-stop], color2, ...


/* ellipse at center, red 0%, blue (default) */
radial-gradient(red, blue);

/* yellow stops at 30% */
radial-gradient(circle at top left, yellow 30%, green);

position

position: static

  • Default
  • Not positioned

position: relative


div {
    position: relative;
    left: 30px;
    top: -25px;
}
  • Positioned relative to its normal position
  • top, right, bottom, left

position: fixed


header {
    position: fixed;
    top: 0;
}
            
body {
    margin-top: 50px;
}
        
h1 {
    // Offset anchor for fixed header
    padding-top: 50px;
    margin-top: -50px;
}
  • Positioned relative to viewport
  • Removes the element from the normal flow
  • top, right, bottom, left

position: absolute


#container {
    position: relative;
}
        
nav {
    position: absolute;
    left: 0;
}
  • Relative to positioned parent container
  • Uses body if no positioned parent
  • Removes the element from the normal flow
  • top, right, bottom, left

position: sticky


header {
    position: sticky;
    top: 30px;
}
  • Relative to scroll position
  • Toggles between relative and fixed
  • Stays in normal flow while relative
  • top, right, bottom, left

overflow

  • overflow
  • overflow-x
  • overflow-y

    overflow: auto;
}
visible
  • default
  • overflow is not clipped
  • overflow renders outside the box
hidden
  • overflow is clipped
  • overflow will be invisible
scroll
  • overflow is clipped
  • scrollbar is always showing
auto
  • overflow is clipped
  • scrollbar only when necessary

transition-property

Properties

transition-timing-function

ease Increases in velocity towards the middle of the transition, slowing back down at the end.
linear Transitions at an even speed.
ease-in Starts off slowly, with the transition speed increasing until complete.
ease-out Starts transitioning quickly, slowing down the transition continues.
ease-in-out Starts transitioning slowly, speeds up, and then slows down again. 

.target {
  font-size: 14px;
  transition: font-size 4s 1s;
}

.target:hover {
  font-size: 36px;
}
