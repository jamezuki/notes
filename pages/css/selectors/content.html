::before / ::after

  • add something before or after a selected element
  • must have a content property

.heart::after {
    background-color: pink;
    content: "";
    border-radius: 50%;
}

Attribute

ele[attr] elements with attr
ele[attr="value"] matchs "value"
ele[attr*="value"] contains "value"
ele[attr^="value"] value preceded by "value"
ele[attr$="value"] value followed by "value"
ele[attr~="value"] "value" is one of its values
element[attr="value"]

file.html


<a href="#selectors-attribute">
    element[attr="value"]
</a>

file.css


a[href="#selectors-attribute"] {
    color: purple;
}

Adjacent

  • Matches the second element only if it immediately follows the first element.
  • Both elements must be children of the same parent.

/* <p> that come immediately after any img */
img + p {
    font-weight: bold;
}

:not()

Represents elements that do not match a list of selectors.


/* Selects any element that is NOT a <p> */
:not(p) {
    color: blue;
}

/* <p> elements that are not .fancy */
p:not(.fancy) {
    color: green;
}

:first-of-type

  • First element of its type among a group of sibling elements
  • Universal selector (*) is implied when no selector is written.

p:first-of-type {
    color: red;
}

.class:nth-child()


/* Selects child 3 of class */
.class:nth-child(3) { }

/* Selects class childs:
  3*0 + 1,
  3*1 + 1,
  3*2 + 1, 
  ... */
.class:nth-child(3n + 1) { }
