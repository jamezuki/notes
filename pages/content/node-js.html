Node Version Manager

For managing multiple node versions.
https://github.com/nvm-sh/nvm
Restart terminal. 


# Must return nvm
command -v nvm

# Install node & npm
nvm install node

versioning

package.json


"package": "^4.17.4"

~ Updates Patch
^ Updates Minor and Patch (Default)
* Updates Major, Minor & Patch
none Installs exact version

Modules

--save-dev

For modules needed only in development.

dotenv Environment variables (credentials)

-g

for modules you want to use in the command line

  • installs on machine
  • not package.json
  • not node_modules
sass CSS preprocessor
nodemon Restarts server when file changes
live server Refresh browser when file changes

require


// built-in or npm modules

// custom module, same dir

// custom module, other dir
var dt = require('/path/module');
