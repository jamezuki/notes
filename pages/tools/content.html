Windows: yt-dlp

  1. Download yt-dlp.exe
  2. Download FFmpeg release essentials
  3. Create yt-dlp directory in C
  4. Copy inside yt-dlp directory:
    • yt-dlp.exe
    • ffmpeg/bin content
    • yt-dlp.bat
    • icon.ico
    • yt-dlp.lnk
  5. Put C:\yt-dlp in PATH
  6. Assign icon to yt-dlp.lnk
  7. Pin yt-dlp.lnk to Start menu
