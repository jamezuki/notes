Primitives

  1. undefined

  4. number

    Arbitrary length

    • May be used as the key of an Object.

Script load

JS must be loaded in his own thread

in file.js


document.addEventListener("DOMContentLoaded", function() { 
    ... 
});

in file.html

defer
executed when the page has been loaded
for dependencies ( use of DOM, etc.. )
executed in order.

<script src="script.js" defer></script>
async
executed independently
no dependencies

<script src="script.js" async></script>

ES6 Modules

export

default export
Only one per module.
Can use any name when imported.
named export
Multiple per module.
Import must use same name as exported.

// Export default
export default myDefault

// Export named
export { myExport1, myExport2 }

import


// Import entire module
import * as myModule from './module.js'

// Import default export
import myDefault from './module.js'

// Import a single export
import {myExport} from './module.js'

// Import multiple exports
import {foo, bar} from './module.js'

// Import with alias
import {name1 as name2} from './module.js'

index.html

type='module' automatically defers script
Only link entry point 


<head>
  ...
  <script type='module' src='main.js'></script>
</head>

data-*

  • Store extra information on tags
  • Associated with element but does not need any defined meaning

Any attribute on any element whose attribute name starts with data- is a data attribute.

Html


<article
    id="myArticle"
    data-columns="3"
    data-index-number="12314">
    ...
</article>

JavaScript access

Dashes are converted to camelCase


const article = document.querySelector('#myArticle');

article.dataset.columns // "3"
article.dataset.indexNumber // "12314"

CSS access


article::before {
    content: attr(data-index-number);
}

article[data-columns='3'] {
    width: 400px;
}

Called when an instance of the element is created or upgraded
connectedCallback()
Called when the element is inserted into de DOM
disconnectedCallback()
Called when the element is removed from the DOM
attributeChangedCallback(attributeName, oldValue, newValue)
Called when an attribute is added, removed, updated or replaced
