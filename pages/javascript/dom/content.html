DOM

Document Object Model

  • Tree-like representation of the contents of a webpage
  • Relationships between it's nodes (Html tags)
  • Browser converts Html to DOM
  • DOM objects can be manipulated with JavaScript

Query Selectors

  • gets html element
  • selector is like css
  • can be used in any element (document is root)

// reference to the first match of selector
let x = document.querySelector(selector);

// nodelist with references to all matches
let y = document.querySelectorAll(selectors);

nodelist to array


// ES6
let z = Array.from(y);

Element Creation

  • creates element in memory

const div = document.createElement('div');

Element Append

  • returns node appended

// appends childNode as the last child of parentNode
let child = parentNode.appendChild(childNode);

// inserts newNode into parentNode before referenceNode
let child = parentNode.insertBefore(newNode, referenceNode);

Element Remove


// removes child from parentNode on the DOM 
let child = parentNode.removeChild(child);

Attribute Edit

Set


div.setAttribute('id', 'theDiv');
// if id exists update it to 'theDiv'
// else create an id with value 'theDiv'
// returns undefined

Get


div.getAttribute('id');
// returns value of specified attribute

Remove


div.removeAttribute('id');
// removes specified attribute
// returns undefined

Class Edit

Add


div.classList.add('new');
// adds class "new" to div

Remove


div.classList.remove('new');
// remove "new" class from div

Toggle


div.classList.toggle('active');
// if div does not have class "active" add it
// if div does have class "active" remove it

Add Text


div.textContent = 'Hello World!'
// creates a text node
// inserts it in div

Add CSS

Use toggle rather than adding and removing CSS.


div.style.cssText = 'color: blue; background: white';
// adds several style rules

Add HTML

  • Use textContent for text
  • Escape characters if it's user input

div.innerHTML = '<span>Hello World!</span>';
// renders the html inside div
