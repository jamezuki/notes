Spread syntax

  • ES6
  • Allows an iterable to be expanded
  • Must be prefixed by ...

let a1 = [2, 3];
let a2 = [1, ...a1, 4, 5];
//  [1, 2, 3, 4, 5]

splice

  • add/remove items by index
  • returns removed items

start index to start
deleteCount number of items to be removed
item1, item2, .. items to be added at start 

let x = [0, 1, 2, 3];

const start = 1;
const deleteCount = 2;

let y = x.splice(start, deleteCount, 5, 6)

// x = [0, 5, 6, 3]
// y = [1, 2]

slice

slice(start, end)

  • returns sliced array
  • end not included in the extracted array
  • if a parameter is negative, the position is counted from the end of the string.
  • if start is omitted starts from 0
  • if end is omitted returns the rest of the string
start index to start
end index to end 

const x = [0, 1, 2, 3, 4];

console.log(x.slice(2));
// [2, 3, 4]
            
console.log(x.slice(2, 4));
// [2, 3]

filter

Creates a new array with the elements that pass the callback condition.


const numbers = [1, 2, 3, 4, 5, 6];

const result = numbers.filter( n => n > 3);

console.log(result); // Array [4, 5, 6]
Callback arguments
currentValue The current element of the array
index (optional) The index of the current element
array (optional) The array of the element
thisArg (optional) Value to use as this

Callback arguments
currentValue The current element of the array

array (optional) The array of the element
thisArg (optional) Value to use as this

sort

No callback

Converts elements to string and sorts alphabetically

For strings only


const months = ['March', 'Jan', 'Feb', 'Dec'];
months.sort();
months; // Array ["Dec", "Feb", "Jan", "March"]

With callback

Callback arguments
a The first element for comparison
b The second element for comparison
Callback return
return > 0 change elements places
return <= 0 do nothing 

const numbers = [1, 30, 4, 21, 100000];

// Ternary
numbers.sort((a, b) => a > b ? 1 : -1);

// Short
numbers.sort((a, b) => a - b);

numbers; // Array[1, 4, 21, 30, 100000]

reduce

Returns the accumulator, whose value is remembered across each iteration

If no initialValue provided, index 0 will be used as accumulator

reduce arguments
callback function to execute on each element
initialValue (optional) accumulator first value
Callback arguments
accumulator accumulates callback's return values
currentValue The current element of the array
index (optional) The index of the current element
array (optional) The array of the element 

const array1 = [1, 2, 3, 4];
const x = array1.reduce((accumulator, current) => {
    return accumulator + current;
}, 0);

x; // 10

Example with object


const data = ['car', 'car', 'truck', 'truck', 'bike', 'walk',
'car', 'van', 'bike', 'walk', 'car', 'van', 'car', 'truck' ];

const transports = data.reduce((obj, item) => {
    if (!obj[item]) obj[item] = 0;
        obj[item]++;
        return obj;
}, {});

console.log(transports); // {car: 5, truck: 3, ... }

some / every

some()

Returns true if at least one element match the condition.

Callback arguments:

  • element
  • index (optional)
  • array (optional)
  • thisArg (optional)

const array = [1, 2, 3, 4, 5];

const even = (element) => element % 2 === 0;

console.log(array.some(even)); // true

every()

Returns true if all elements match the condition.

Callback arguments:

  • element
  • index (optional)
  • array (optional)
  • thisArg (optional)

const array = [1, 2, 3, 4, 5];

const even = (element) => element % 2 === 0;

console.log(array.every(even)); // false

find

Returns the first element that matches the condition.

Callback arguments:

  • element
  • index (optional)
  • array (optional)
  • thisArg (optional)

const array1 = [5, 12, 8, 130, 44];

const found = array1.find(element => element > 10);

console.log(found); // 12

findIndex

Returns the index of the first element that matches the condition.

Returns -1 if no matches.

Callback arguments:

  • element
  • index (optional)
  • array (optional)
  • thisArg (optional)

const array1 = [5, 12, 8, 130, 44];

const x = (element) => element > 13;

console.log(array1.findIndex(x)); // 3

indexOf

Returns the index of the first element that matches the value.

  • Returns -1 if no matches
  • No callback

Arguments:

  • searchElement
  • fromIndex (optional)

const beasts = ['ant', 'bison', 'camel', 'duck', 'bison'];

console.log(beasts.indexOf('bison')); // 1

console.log(beasts.indexOf('bison', 2)); // 4

Spread syntax

Expands an iterable.


const sum = (x, y, z) => x + y + z
const numbers = [1, 2, 3]

console.log(sum(...numbers)) // 6

Destructuring

Unpack values into variables.
Based on position in the array. 


const alphabet = ['A', 'B', 'C', 'D', 'E', 'F']
const numbers = [1, 2, 3, 4, 5, 6]

Assign


const [a, b] = alphabet 
console.log(a) // 'A'
console.log(b) // 'B'

Skip


const [a,, c] = alphabet
console.log(a) // 'A'
console.log(c) // 'C'

Spread Operator


const [a,, c, ...rest] = alphabet
console.log(a) // 'A'
console.log(c) // 'C'
console.log(rest) // ['D', 'E', 'F']

Combine


const combined = [...alphabet, ...numbers]

Default values


function sumAndMultiply(a, b) {
	return [a+b, a*b]
}

const [sum, mul, div = 'no division'] = sumAndMultiply(2, 3)
console.log(sum) // '5'
console.log(mul) // '6'
console.log(div) // 'no division'
